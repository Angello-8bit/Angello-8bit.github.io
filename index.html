01 OCR

(Python)

Optical character recognition. Using python libraries such as cv2, pytesseract, PIL, and numpy. After multiple days of unit testing and optimization using Thresholding, noise reduction, dilation, and grayscale. I was able to work on different types of images and extract text from old pdfs.

02 Webapp

(Python/js/hmtl/css/api/sql)

Using Flask, os, SQLAlchemy, and spotify api(spotipy), I created a web application that could receive the name of a song and give back ten songs that are statically correlated to the one inputted. The output is trained on the data of 10,000 different songs and using features given by spotify into what makes a song what it is like tempo, danceability, duration, etc.

03 Web Scraper

(Python)

Using python libraries such as beautifulsoup, requests, Urllib, and Scrapy. I was able to extract information from different wiki pages and store them in a csv file.

04 Word document creator

(Python)

Using python libraries such as docx, google, and multiple web scraping libraries. I created a self arranged word document that could have information gathered from web scraping and google searches.

05 spring boot[Java Framework]

(java/sql/django/docker)

Using a spring boot template from start.spring.io and using sql+django+docker to gather and save data. A java framework was formed that allowed the user to login and store information about themselves.

06 Backend Microservices

(Golang/Api/sql)

Using a rested Api that only provides https endpoints and has timestamps, also using django+sql+docker. A microservice that allows the user to buy different colored shapes, manage their purchases and like and save products.

07 Unreal Engine 5 game [WIP]

(C++)

Learning to Create a game that is a turn based game similar to wargroove and have it become a 2d fighter during the conflict.
